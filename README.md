# Python Core - Portfolio Programs

Each file is a separate program for a Python Core topic. All text and comments are in English. Use hyphen (-) only, no em/en dashes.

| File | Topic | Description |
|------|--------|-------------|
| `01_if_else_bmi_calculator.py` | **if/else, conditions** | BMI calculator with categories and health recommendations |
| `02_loops_password_generator.py` | **Loops (for, while)** | Password generator and strength checker |
| `03_files_expense_tracker.py` | **Files** | Expense tracker: add, save to file, summary by category |
| `04_functions_guess_number.py` | **Functions** | Guess the number game: functions for secret, input, round, win count |
| `05_lists_dicts_contact_book.py` | **Lists and dicts** | Contact book: add, search, list, delete |
| `06_strings_text_analyzer.py` | **Strings** | Text analyzer: palindrome, stats, Caesar cipher |
| `07_exceptions_safe_calculator.py` | **Exceptions (try/except)** | Calculator with input validation and division-by-zero handling |
| `08_oop_mini_bank.py` | **OOP (classes)** | Mini bank: account class with deposit, withdraw, balance, history |
| `09_data_structures_sets_tag_manager.py` | **Sets** | Unique tag manager: add, remove, union, intersection, difference |
| `10_data_structures_tuples_leaderboard.py` | **Tuples** | Score leaderboard: immutable (name, score) records, top N |
| `11_data_structures_collections_task_queue.py` | **collections** | Task queue with undo: deque (FIFO), Counter for stats |
| `12_types_builtin_and_conversion.py` | **Built-in types** | int, float, complex, bool, str; type(); conversion; no mixing types in math |
| `13_types_bytes_bytearray.py` | **bytes, bytearray** | Encode/decode, immutable bytes vs mutable bytearray |
| `14_operations_logical_membership.py` | **Logical and membership** | and, or, not; in, not in (strings, lists, dicts, sets) |
| `15_operations_bitwise.py` | **Bitwise** | AND, OR, XOR, NOT, left/right shift |
| `16_data_structures_frozenset.py` | **frozenset** | Immutable set; frozenset as dict key |
| `17_library_json.py` | **json** | json.dumps/loads, json.dump/load (files) |
| `18_library_csv.py` | **csv** | csv.writer, csv.reader, csv.DictReader |
| `19_library_regex.py` | **re** | re.search, re.findall, re.match, groups |
| `20_library_pathlib.py` | **pathlib** | Path, cwd, glob, read_text, write_text |
| `21_library_datetime.py` | **datetime** | datetime, strftime, strptime, timedelta |

## Run

```bash
python 01_if_else_bmi_calculator.py
python 02_loops_password_generator.py
# etc.
```

All programs are in English and run in the console (some are demos, some interactive).
